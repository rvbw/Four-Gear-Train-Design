# Four-Gear Train Design with 1:18 Gear Ratio

![Four Gear Train Assembly](images/assembly.png)

## Overview
This project presents the **design and mechanical assembly of a four-gear train system**.  
All gears were designed from scratch and assembled to achieve a final transmission ratio of **1:18** between the first and fourth gears.

The system consists of four spur gears mounted on parallel shafts.

## Project Purpose
The objective of this project is to:
- Design a multi-stage gear transmission system.
- Achieve a specific overall gear ratio (1:18).
- Understand rotational speed reduction and torque amplification.
- Practice gear meshing alignment and shaft positioning.

This project was completed for learning and mechanical design practice purposes.

## System Configuration
- Gear 1: Input gear
- Gear 2: Intermediate gear
- Gear 3: Intermediate gear
- Gear 4: Output gear
- Parallel shafts and supports

The intermediate gears (Gear 2 and Gear 3) transmit motion and help achieve the required ratio.

## Gear Ratio

Overall Gear Ratio:

Output Speed / Input Speed = 1 / 18

This means:
- The fourth gear rotates 18 times slower than the first gear.
- Torque is theoretically multiplied by 18 (neglecting losses).

## Assembly Scope
- Custom design of all four spur gears.
- Proper tooth meshing alignment.
- Shaft positioning and spacing.
- Full mechanical assembly without physical testing.

## Mechanical Principle
The total gear ratio in a multi-stage gear train is calculated by multiplying the individual stage ratios.

This system demonstrates controlled speed reduction through sequential gear engagement.
